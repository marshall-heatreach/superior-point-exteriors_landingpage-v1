# Superior Point Exteriors Landing Page

A pixel-perfect landing page built with Astro, Tailwind CSS, and React.

## Features

- **Astro** for fast, static site generation
- **Tailwind CSS** for styling
- **React** for interactive components (quiz form)
- Fully responsive design
- Modern, clean UI

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:4321` to see your site.

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```
